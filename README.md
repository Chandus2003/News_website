News Website using News API
This project is a simple news website that utilizes the News API to fetch and display news articles from various sources.

Overview
This web application fetches news articles using the News API and presents them in a user-friendly interface. Users can browse through different categories of news, 
search for specific topics, and read detailed articles.

Features
Fetches news articles from the News API.
Displays articles by category (e.g., general, technology, sports, etc.).
Allows users to search for articles by keyword.

Technologies Used
HTML5
CSS3
JavaScript
News API
